# WatersysPro Presentation Bundle

## Files
- `WatersysPro_Presentation.html` - technical interactive presentation (engineering focus).
- `WatersysPro_Presentation_Executive.html` - executive interactive presentation (KPI, risks, roadmap).

## Which file to send
1. For technical team / integrators: `WatersysPro_Presentation.html`.
2. For management / business stakeholders: `WatersysPro_Presentation_Executive.html`.

## How to use
1. Open the needed HTML file in any modern browser (Chrome/Edge/Firefox).
2. Use left sidebar for quick navigation.
3. Use search in sidebar to filter sections.
4. Expand details blocks where available.

## Sharing by email
- Attach only one HTML file based on audience.
- The file is self-contained and opens offline.

## Optional PDF export
- Open HTML in browser.
- Print -> Save as PDF.
- Обзор проекта и архитектуры
- Прошивка и FSM
- Подробное описание обычного меню
- Подробное описание инженерного меню
- Backend/Mobile часть, demo-сценарий, риски и документация
