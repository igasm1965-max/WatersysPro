WaterSysPro — система управления водоочисткой

Full-stack решение: ESP32 firmware + Node.js backend + React Native mobile, с поддержкой MQTT, web UI, журналирования и сервисных режимов.

ESP32 MQTT + Web UI Node.js + Docker React Native

1. Обзор проекта

Назначение

Автоматизация цикла водоочистки с удаленным мониторингом и управлением.

Три уровня системы

  • Прошивка ESP32: FSM, реле, датчики, web, MQTT
  • Backend Node.js: REST API, WebSocket, Docker
  • Mobile React Native: login, dashboard, settings
Статус
Active
Релизная база
1.1.0
Контур
ESP32 + Cloud

2. Архитектура и стек

Поток данных

ESP32 собирает телеметрию, FSM формирует состояние, данные отдаются через MQTT и web/API, клиенты backend/mobile отображают состояние и отправляют команды.

Firmware

  • PlatformIO + Arduino
  • Web UI на устройстве
  • MQTT telem + cmd
  • Логирование и сервисные режимы

Application

  • Backend Node.js + WebSocket
  • Docker Compose инфраструктура
  • Mobile React Native + EAS build

3. Прошивка и FSM

Ключевые возможности прошивки

  • Управление 7 реле
  • Два ультразвуковых датчика уровня
  • FSM для рабочих циклов водоочистки
  • MQTT и web управление
  • Текстовый журнал событий
FSM контур

IDLE -> FILLING -> OZONATION -> AERATION -> SETTLING -> FILTRATION -> BACKWASH

Критичный сценарий эксплуатации меню
  1. Ввод базовых таймеров и уровней баков в обычном меню.
  2. Переход в Engineer Menu и настройка safety timeouts.
  3. Проверка Pump Protection на тестовом запуске без нагрузки.
  4. Настройка Wi-Fi/MQTT, затем Log Filter под нужный объем телеметрии.
  5. Фиксация конфигурации и контрольный прогон FSM.
