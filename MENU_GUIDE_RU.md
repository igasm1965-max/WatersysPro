# 📋 Руководство по работе с меню WaterSysPro

## Обзор системы меню

Система меню WaterSysPro предоставляет интуитивный интерфейс для настройки всех параметров системы управления водоочисткой. Меню реализовано как многоуровневая иерархическая структура с навигацией через энкодер (поворот для выбора, нажатие для подтверждения).

## 📱 Основные элементы управления

### Энкодер (кнопка + вращение)
- **Вращение вправо/влево** - перемещение по пунктам меню
- **Короткое нажатие** - выбор текущего пункта / подтверждение действия
- **Длительное нажатие** (в некоторых контекстах) - возврат на уровень выше

### Дисплей (LCD 20x4)
- Отображает до 4 пунктов меню одновременно
- Текущий выбранный пункт выделяется символом `>`
- Прокрутка при превышении количества пунктов

## 🏠 Структура главного меню

Главное меню содержит следующие пункты (в порядке отображения):

1. **Setting up timers** - Настройка таймеров
2. **Cleaning the filter** - Очистка фильтра
3. **Setting tank level** - Установка уровней баков
4. **Manual operation** - Ручное управление
5. **Date & Time** - Дата и время
6. **System Logging** - Системное логирование
7. **Schedule Settings** - Настройки расписания
8. **Night Mode** - Ночной режим
9. **Reset Settings** - Сброс настроек
10. **Engineer Menu** - Инженерное меню (требует пароль)
11. **EXIT** - Выход из меню

## 🔧 Детальное описание пунктов меню

### 1. Настройка таймеров (Setting up timers)
Включает настройку продолжительности различных процессов:
- **Setling Water timer** - Таймер отстаивания воды
- **Aeration timer** - Таймер аэрации
- **Ozonation timer** - Таймер озонации
- **Backlight timer** - Таймер подсветки дисплея
- **Back** - Назад

**Как использовать:** Выберите таймер, введите значение в секундах с помощью энкодера, подтвердите.

### 2. Очистка фильтра (Cleaning the filter)
Настройка параметров промывки фильтра:
- **Filter wash timer** - Таймер промывки фильтра
- **Wash period** - Периодичность промывки (интервал)
- **Back** - Назад

### 3. Установка уровней баков (Setting tank level)
Настройка минимального и максимального уровней для каждого бака:
- **Tank 1** - Бак 1 (минимальный/максимальный уровень в см)
- **Tank 2** - Бак 2 (минимальный/максимальный уровень в см)
- **Back** - Назад

### 4. Ручное управление (Manual operation)
Прямое управление реле и процессами:
- Включение/выключение отдельных реле
- Запуск конкретных процессов вручную
- Просмотр текущего состояния системы

### 5. Дата и время (Date & Time)
Настройка системных часов:
- **Set Current Time** - Установка текущего времени (часы, минуты, секунды)
- **Set Current Date** - Установка текущей даты (день, месяц, год)
- **Set Time Zone** - Установка часового пояса
- **View Date/Time** - Просмотр текущей даты и времени
- **Back** - Назад

### 6. Системное логирование (System Logging)
Управление журналом событий:
- **Event Logs** - Журнал событий (подменю)
- **Clear Event Logs** - Очистка всех логов событий
- **Prune Event Logs** - Удаление старых логов
- **Format SD Card** - Форматирование SD карты
- **Back** - Назад

#### Подменю Event Logs:
- **View Event Logs** - Просмотр логов событий
- **Event Statistics** - Статистика событий
- **Event Graph** - График событий
- **Filter by Type** - Фильтрация по типу
- **Clear Old Events** - Очистка старых событий
- **Back** - Назад

### 7. Настройки расписания (Schedule Settings)
Настройка временных интервалов работы системы:
- **Set Start Time** - Установка времени начала работы
- **Set End Time** - Установка времени окончания работы
- **Enable/Disable** - Включение/выключение расписания
- **Back** - Назад

### 8. Ночной режим (Night Mode)
Настройка ночного режима работы:
- **Set Start Time** - Установка времени начала ночного режима
- **Set End Time** - Установка времени окончания ночного режима
- **Enable/Disable** - Включение/выключение ночного режима
- **Back** - Назад

### 9. Сброс настроек (Reset Settings)
Восстановление заводских настроек:
- Сбрасывает все настройки к значениям по умолчанию
- **Требует подтверждения** (Yes/No)
- Не затрагивает журнал событий и данные на SD карте

### 10. Инженерное меню (Engineer Menu) 🔒
Расширенные настройки, защищенные паролем (по умолчанию: `123456`):

#### Подменю инженерного меню:
- **Safety Timeouts** - Таймауты безопасности
- **Watchdog Settings** - Настройки Watchdog
- **Change Password** - Смена пароля инженерного меню
- **System Info** - Системная информация
- **Network Info** - Сетевая информация
- **Log Filter** - Фильтр логов
- **Wi-Fi Setup** - Настройка Wi-Fi
- **MQTT Setup** - Настройка MQTT
- **Factory Reset (!)** - Полный сброс к заводским настройкам
- **Back** - Назад

#### Safety Timeouts (Таймауты безопасности):
- **Filling Timeout** - Таймаут заполнения
- **Filtration Timeout** - Таймаут фильтрации
- **Pump Dry Timeout** - Таймаут сухого хода насоса
- **Pump Min Delta** - Минимальная дельта уровня для насоса
- **Pump Dry Consec.** - Количество подряд проверок сухого хода
- **Sensor Poll Period** - Период опроса датчиков
- **Back** - Назад

**Примечание:** Таймауты озонации, аэрации, отстаивания и обратной промывки были удалены как дублирующие функции основных счетчиков времени процессов. Безопасность этих процессов теперь контролируется через основные таймеры продолжительности (ozonationDuration, aerationDuration, setlingDuration, filterWashDuration).

#### Watchdog Settings:
- **Enable/Disable** - Включение/выключение Watchdog
- **Timeout (sec)** - Таймаут Watchdog в секундах
- **Back** - Назад

### 11. Выход (EXIT)
Завершение работы с меню и возврат к основному экрану системы.

## 🔐 Инженерное меню: доступ и безопасность

### Пароль по умолчанию
- **Стандартный пароль**: `123456`
- **Рекомендуется изменить** при первом входе через пункт "Change Password"

### Процедура входа
1. Выберите "Engineer Menu" в главном меню
2. Введите 6-значный пароль с помощью энкодера
3. При успешном вводе откроется инженерное меню
4. При ошибке - возврат в главное меню

### Блокировка доступа
- Инженерное меню автоматически блокируется после выхода
- Для повторного входа требуется снова ввести пароль
- После 3 неудачных попыток возможна временная блокировка

## ⚙️ Пункт "Перевод" (Translation)

**Важное примечание:** В текущей версии системы (v1.1.0) пункт "Перевод" отсутствует в меню. Возможные интерпретации:

### 1. Перевод интерфейса
- Система в настоящее время использует **английский язык** интерфейса
- Русскоязычные версии текстов могут быть добавлены в будущих обновлениях
- Для перевода потребуется:
  - Создание системы локализации
  - Хранение строк на разных языках
  - Добавление пункта меню для выбора языка

### 2. Перевод данных
- Возможно относится к преобразованию единиц измерения
- Или к передаче данных между системами (протоколы)

### 3. Резервное копирование/восстановление
- В некоторых системах "перевод" означает перенос настроек между устройствами

**Рекомендация:** Если требуется функционал смены языка интерфейса, его необходимо реализовать как:
1. Новый пункт в меню "Language" или "Язык"
2. Хранение строк в отдельном файле локализации
3. Механизм динамического переключения языка

## 🛠️ Практические советы по работе с меню

### Навигационные сочетания
- **Быстрый возврат**: Длительное нажатие кнопки энкодера (в большинстве подменю)
- **Пропуск пунктов**: Быстрое вращение энкодера
- **Точная настройка**: Медленное вращение для пошагового изменения значений

### Ввод числовых значений
1. Выберите параметр для настройки
2. Вращайте энкодер для изменения значения
3. Нажмите для подтверждения
4. Для отмены - выйдите без подтверждения

### Работа с подменю
- Каждый уровень меню имеет пункт "Back" для возврата
- Максимальная глубина вложенности: 3 уровня
- Текущий путь отображается в верхней строке дисплея

## ⚠️ Предупреждения и ограничения

### Критические настройки
- **Factory Reset** - безвозвратно удаляет все пользовательские настройки
- **Format SD Card** - уничтожает все данные на карте памяти
- **Change Password** - при утере пароля потребуется сброс к заводским настройкам

### Ограничения по времени
- Некоторые таймауты имеют минимальные/максимальные значения
- При вводе недопустимых значений система автоматически корректирует их

### Безопасность
- Инженерное меню защищено паролем
- Рекомендуется регулярно менять пароль
- Не используйте простые комбинации (123456, 000000)

## 🔄 Процедуры восстановления

### Забыт пароль инженерного меню
1. Выполните аппаратный сброс (кнопка RESET на плате)
2. При запуске удерживайте кнопку энкодера
3. Система предложит сброс пароля к значению по умолчанию

### Ошибочные настройки
1. Используйте пункт **Reset Settings** для сброса обычных настроек
2. Для полного сброса используйте **Factory Reset** в инженерном меню
3. Предварительно сделайте резервную копию важных данных

### Зависание в меню
1. Нажмите и удерживайте кнопку энкодера 5 секунд
2. Система выполнит принудительный выход из меню
3. При повторных проблемах - перезагрузите устройство

## 📊 Логирование действий в меню

Все изменения, сделанные через меню, регистрируются в журнале событий:
- **Тип события**: `MENU_SETTING_CHANGED`
- **Параметры**: Название параметра, старое значение, новое значение
- **Временная метка**: Дата и время изменения

Для просмотра:
1. Перейдите в **System Logging** → **Event Logs** → **View Event Logs**
2. Используйте фильтр по типу `MENU_`

## 📞 Поддержка и устранение неисправностей

### Частые проблемы и решения

| Проблема | Возможная причина | Решение |
|----------|-------------------|---------|
| Меню не реагирует на энкодер | Проблема с аппаратной частью | Проверьте подключение энкодера |
| Пункты меню отображаются некорректно | Повреждение данных в памяти | Выполните сброс настроек |
| Не могу войти в инженерное меню | Неверный пароль | Используйте пароль по умолчанию (1234) |
| Значения сбрасываются после перезагрузки | Проблема с сохранением в EEPROM | Проверьте батарейку RTC |
