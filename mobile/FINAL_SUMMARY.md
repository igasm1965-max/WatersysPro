## 🚀 НАЧНИТЕ ОТСЮДА:

### Вариант 1: Быстро (3 команды)

**Шаг 1 - Установить инструменты (один раз):**
```bash
npm install -g eas-cli
```

**Шаг 2 - Авторизоваться (один раз):**
```bash
eas login
```

**Шаг 3 - Собрать APK (может повторяться):**
```bash
cd d:\WatersysPro\mobile
eas build -p android
```

✅ Подождите 5-15 минут, получите ссылку на скачивание APK!

---

### Вариант 2: Через батник (для Windows)

1. Откройте `d:\WatersysPro\mobile\build-apk.bat`
2. Выберите опцию 1 (Cloud Build)
3. Готово! ✅

## 🎯 ПОСЛЕ СБОРКИ APK:

1. **Скачайте** APK файл по ссылке в консоли
2. **Передайте** на Android устройство
3. **На устройстве:**
   - Включите: Настройки → Безопасность → Установка из неизвестных источников
   - Откройте APK файл
   - Подтвердите установку
4. **Запустите** приложение! 🎉
