## 🚀 БЫСТРЫЙ СТАРТ (копируйте команды)

### Шаг 1 - Один раз: Установить EAS
```bash
npm install -g eas-cli
```

### Шаг 2 - Один раз: Авторизоваться
```bash
eas login
```
(Откроется браузер для создания/входа в Expo)

### Шаг 3 - Собрать APK (может повторяться)
```bash
cd d:\WatersysPro\mobile
eas build -p android
```

✅ **Готово!** Подождите 5-15 минут, получите ссылку на скачивание APK.

## 📱 После того как у вас есть APK

1. **Скачайте** APK файл по ссылке из консоли
2. **Передайте** на Android устройство (USB, облако, Email и т.д.)
3. **На устройстве** включите: Настройки → Безопасность → Установка из неизвестных источников
4. **Откройте** APK файл на устройстве
5. **Подтвердите** установку
6. **Запустите** приложение! 🎉

## 🔧 Настройка перед сборкой (если нужно)

### Изменить IP адрес сервера

Если ваш сервер не на `192.168.3.13`, отредактируйте:

**Файл:** `src/services/api.ts`
```typescript
// Измените эту строку:
const API_URL = 'http://192.168.3.13:80/api';

// На вашу:
const API_URL = 'http://192.168.1.100:3000/api';
```

### Изменить версию приложения

**Файл:** `app.json`
```json
{
  "version": "1.0.0",  ← Измените версию
  "android": {
    "versionCode": 1   ← И код версии
  }
}
```

## ❓ Часто задаваемые вопросы

**Q: Зачем облачная сборка?**
A: Не нужно устанавливать 10GB Android SDK. Все делается на серверах Expo.

**Q: Сколько времени займет сборка?**
A: 5-15 минут первый раз, затем быстрее.

**Q: Нужна ли мне учетная запись где-то?**
A: Да, создайте бесплатный аккаунт на https://expo.dev/

**Q: Могу ли я собрать без интернета?**
A: Облачная нет. Для локальной сборки нужен Android SDK.

**Q: Как обновить приложение?**
A: Измените версию в app.json, затем eas build -p android

**Q: Где скачать APK после сборки?**
A: На https://dashboard.expo.dev или по ссылке в консоли
