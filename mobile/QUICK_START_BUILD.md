## 🎯 快快 Начните здесь - Облачная сборка (самый простой способ)

### Шаг 1 - Установка EAS (один раз)
```bash
npm install -g eas-cli
```

### Шаг 2 - Авторизация (один раз)
```bash
eas login
```
Откроется браузер - создайте аккаунт или авторизуйтесь на expo.dev

### Шаг 3 - Сборка APK 🎬
Перейдите в папку проекта и выполните:
```bash
cd d:\WatersysPro\mobile
eas build -p android
```

Сборка займет 5-15 минут. Вам будет предложено:
- Выбрать тип APK (выберите `apk` для обычной установки)
- Выбрать профиль кода подписа (выберите `default`)

После завершения вы получите ссылку на скачивание APK файла! ✅

## 🔄 Другие варианты сборки

### Локальная сборка (требует Android SDK)
```bash
eas build -p android --local
```

### Просмотр статуса всех сборок
```bash
eas build:list
```

### Просмотр логов текущей сборки
```bash
eas build:view <BUILD_ID>
```

## 📱 Установка APK на устройство

1. **Скачайте** APK файл со ссылки из консоли
2. **Передайте** на Android устройство (через USB, облако и т.д.)
3. **Откройте** APK файл на устройстве
4. **Подтвердите** установку

⚠️ **Важно**: У вас должно быть разрешено установка из неизвестных источников:
- Настройки → Безопасность → Установка из неизвестных источников (ВКЛ)

## ❓ Часто встречаемые проблемы

### "Command not found: eas"
```bash
npm install -g eas-cli
```

### "Not authenticated"
```bash
eas logout
eas login
```

### "Cannot connect to API"
Проверьте интернет соединение, затем повторите:
```bash
eas build -p android
```

### Где скачать APK?
После окончания сборки - в консоли будет ссылка или перейдите на dashboard.expo.dev в сборки проекта.
