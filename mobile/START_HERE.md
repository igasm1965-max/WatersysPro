# 🎯 РЕЗЮМЕ: Как создать APK за 5 минут

## ⚡ Самый быстрый способ

**Просто выполните в PowerShell/CMD в папке `d:\WatersysPro\mobile`:**

### Шаг 1️⃣ - Установить инструменты (один раз)
```bash
npm install -g eas-cli
```

### Шаг 2️⃣ - Авторизоваться (один раз)
```bash
eas login
```
(Откроется браузер, создайте/авторизуйтесь на expo.dev)

### Шаг 3️⃣ - Собрать APK 
```bash
eas build -p android
```

✅ **Это все!** Ждите 5-15 минут, получите ссылку на скачивание APK.

---

## 🪟 Для Windows - используйте батник

```bash
cd d:\WatersysPro\mobile
.\build-apk.bat
```

Выберите опцию 1 (Cloud Build) - это самый надежный способ.

---

## 📱 После скачивания APK

1. Передайте APK на Android устройство
2. На устройстве: Настройки → Безопасность → Установка из неизвестных источников (ВКЛ)
3. Откройте APK файл и подтвердите установку
4. Запустите приложение! 🚀

---

## 🧪 Тестирование ДО сборки APK (рекомендуется)

Запустите в Expo Go (без APK):
```bash
npm start
```

Отсканируйте QR-код в приложении **Expo Go** на телефоне.

## ℹ️ Почему облачная сборка (eas build) - лучший выбор?

✅ Не нужно устанавливать Android SDK (10GB+!!)
✅ Не нужно настраивать Java, Gradle и т.д.  
✅ Автоматическое подписание
✅ Работает из любого места
✅ Быстро и надежно

## ❓ Вопросы?

**"Зачем нужна облачная сборка?"**
- Не нужно устанавливать 10GB Android SDK
- Сборка быстрее и надежнее
- Автоматическое подписание сертификатов

**"Когда мне нужна локальная сборка?"**
- Если нужны пользовательские native модули
- Если хотите собирать без интерната (offline)
- Для отладки native кода

**"Где мой скачанный APK?"**
- Dashboard: https://dashboard.expo.dev
- Или прямая ссылка в консоли при eas build

**"Как обновлять приложение?"**
- Просто измените версию в app.json
- Собери новый APK через eas build
- Распространяйте новый APK

## 🔄 Команды которые могут понадобиться

```bash
# Просмотр всех сборок
eas build:list

# Просмотр одной сборки
eas build:view <ID>

# Отмена сборки
eas build:cancel <ID>

# Очистить и переустановить
rm -r node_modules package-lock.json
npm install

# Проверить что все работает
npm start
```
